Bem Vindo(a) ao jogo de Dama

Esse jogo trabalhará matematica com o conceito de MARGIN!

Funcionamento do jogo:

  • Peças Pretas para avancar adcionam um "margin-top"
  • Peças Brancas para avancar adcionam um "margin-bottom"
  • Peças Pretas para andar para a DIREITA adcionam um "margin-left" POSITIVO
  • Peças Pretas para andar para a ESQUERDA adcionam um "margin-left" NEGATIVO
  • Peças Brancas para andar para a DIREITA adcionam um "margin-right" NEGATIVO
  • Peças Brancas para andar para a ESQUERDA adcionam um "margin-right" POSITIVO
  • O tabuleiro tem 8/8 de tamanho, com cada casa tendo 80px de largura e altura

Regras do jogo:

  • Cada jogador tem direito a apenas uma jogada.
  • As peças só podem ser movidas para frente (exeto as damas).
  • As peças só podem andar uma casa para frente por vez (exeto as damas).
  • Única condição para a peça mover para o lado, e pular mais de uma casa, é para "comer" a(s) peça(s) adversária(s).
  • "Comer" a peça para trás vale apenas em combos(mais de 1 peça de uma vez) e sendo a primeira para frente.
  • Ao fazer um combo você deve mover a peça ao detino final.
  • Ao ter uma peça "comida", o jogador que perdeu a peça deverá retira-la do tabuleiro (peças brancas a direta e peças pretas a esquerda)
  • Uma peça vira dama ao chegar do outro lado do tabuleiro
